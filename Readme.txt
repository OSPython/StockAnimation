This script will do the following:
1. Uses PyQt for user input and output display
2. Gets stock info using yfinance
3. Creates a movie
4. Allows user to select the movie play speed.
5. Allows option for saving the movie to disk. 

You'll need the following libraries:
1. PyQt6
2. matplotlib
3. yfinance 
4. pandas 
5. numpy 
6. datetime
